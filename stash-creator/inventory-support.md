# Inventory Support

## Inventory Support

### Supported Inventory Systems

* ox\_inventory
* qb-inventory
* qs-inventory
* ESX inventories

***

### Automatic Detection

Example:

```lua
Config.Inventory = 'auto'
```

***

### Manual Configuration

Example:

```lua
Config.Inventory = 'ox_inventory'
```

***

### Supported Values

```txt
ox_inventory
qb-inventory
qs-inventory
esx
auto
```

***

### Notes

Automatic inventory detection is recommended for most servers.
