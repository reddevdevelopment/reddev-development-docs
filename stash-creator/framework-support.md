# Framework Support

##

### Supported Frameworks

REDDEV Stash Creator supports automatic framework detection.

***

### Supported Frameworks

* QBCore
* QBox
* ESX

***

### Configuration

Example:

```lua
Config.Framework = 'auto'
```

***

### Manual Framework Setup

Example:

```lua
Config.Framework = 'qbcore'
```

Supported values:

```txt
qbcore
qbox
esx
auto
```

***

### Notes

Automatic detection is recommended unless your server requires manual configuration.
