# Troubleshooting

## Troubleshooting

### Stashes Not Saving

#### Check:

* Resource has write permissions
* stashes.json exists
* No console errors are present

***

### Inventory Not Opening

#### Verify:

* Supported inventory is installed
* Inventory started before stash creator
* Correct inventory configured

***

### Marker Not Showing

#### Check:

* Marker enabled in configuration
* Correct coordinates
* Distance settings

***

### Permissions Not Working

#### Verify:

* Jobs configured correctly
* Framework loaded correctly
* Permissions setup properly

***

### Recommended

Always restart the resource after making major configuration changes.
