# Support

## Support

Need help with a REDDEV resource?

The REDDEV Development team provides support for installations, configuration issues, framework compatibility, and troubleshooting.

***

### Support Includes

* Installation help
* Error troubleshooting
* Inventory support
* Framework setup
* Permission setup
* Configuration assistance

***

### Before Opening Support Tickets

Please include:

* Console errors
* Framework used
* Inventory used
* Screenshots if possible
* Steps to reproduce the issue
