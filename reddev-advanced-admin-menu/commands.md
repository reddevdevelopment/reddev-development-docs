# Commands

## Commands

### Main Commands

| Command   | Description            |
| --------- | ---------------------- |
| `/admin`  | Opens the admin menu   |
| `/noclip` | Toggle noclip          |
| `/staff`  | Toggle staff mode      |
| `/dv`     | Delete nearby vehicles |
| `/revive` | Revive a player        |
| `/heal`   | Heal a player          |
| `/bring`  | Bring a player         |
| `/goto`   | Teleport to a player   |

***

### Notes

Some commands may depend on your configured permissions and framework.
