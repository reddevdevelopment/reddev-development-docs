# FiveManage Integration

## FiveManage Integration

### Overview

The REDDEV Advanced Admin Menu supports optional FiveManage integration for advanced media handling and screenshot storage.

***

### Features

* Screenshot uploads
* Media storage
* Evidence collection
* Staff monitoring support

***

### Configuration

Example:

```lua
Config.FiveManage = {
    enabled = true,
    token = 'YOUR_TOKEN_HERE'
}
```

***

### Notes

A valid FiveManage token is required for integration features to function correctly.

***

### Recommended

FiveManage is recommended over local screenshot storage for better performance and easier media management.
