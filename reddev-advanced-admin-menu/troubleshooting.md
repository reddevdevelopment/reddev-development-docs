# Troubleshooting

## Troubleshooting

### Admin Menu Does Not Open

#### Check:

* Resource is ensured
* Permissions are configured
* Dependencies are started
* No console errors are present

***

### Permissions Not Working

#### Verify:

* ACE permissions are added correctly
* Framework groups are configured
* Identifiers are correct

***

### Screenshot Errors

#### Common Causes:

* screenshot-basic not started
* Invalid filesystem permissions
* Incorrect save paths

***

### Database Errors

#### Verify:

* oxmysql is running
* Database connection is configured
* Required tables exist

***

### UI Problems

#### Check:

* Cache is cleared
* Resource restarted
* Browser/NUI errors in console

***

### Recommended

Always use updated server artifacts and latest dependency versions.
